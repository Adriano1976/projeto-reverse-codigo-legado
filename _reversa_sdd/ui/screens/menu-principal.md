# Tela: Menu Principal

## Descrição
Interface central de navegação composta por um banner informativo e um grid de botões de ação.

## Elementos de Interface

### Banner Informativo
- **Tipo:** Container Azul (`SteelBlue`)
- **Conteúdo:** Texto de boas-vindas com fonte Verdana ~15px.
- **Função:** Contextualizar o jogo para o usuário.

### Menu de Topo
- **Tipo:** Menu Bar
- **Opções:** "Arquivo" (Contém comando para fechar/ajuda).

### Grid de Referências
- **Layout:** Grid densa de botões.
- **Botão:** 
  - Texto: Referência Bíblica (ex: "Daniel 1:1-6")
  - Estilo: BG `#B22222` (FireBrick), FG White, Font Verdana.
  - Ação: Abre modal de pergunta correspondente.

## Estados da Tela
- **Normal:** Exibe todos os botões ativos.
- **Interação:** O botão muda levemente de aparência ao ser pressionado (comportamento padrão do Tkinter).
