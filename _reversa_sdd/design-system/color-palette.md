# Paleta de Cores — projeto-python-biblia

As cores são definidas diretamente no código-fonte via nomes de cores padrão do X11 (suportados pelo Tkinter).

| Nome (Token) | Valor (Hex/X11) | Uso Semântico | Confiança |
|--------------|-----------------|---------------|-----------|
| `primary-bg` | `SteelBlue` | Fundo do banner principal | 🟢 |
| `button-bg` | `FireBrick` | Fundo das referências bíblicas | 🟢 |
| `success-bg` | `green` | Feedback positivo | 🟢 |
| `error-bg` | `red` | Feedback negativo | 🟢 |
| `option-bg` | `Khaki` | Botões de alternativas (questões) | 🟢 |
| `text-primary` | `white` | Texto sobre fundos escuros | 🟢 |
| `text-secondary` | `black` | Texto padrão | 🟢 |

## Escala de Neutros
- **Branco:** Utilizado para legibilidade em botões e banners.
- **Preto:** Utilizado em menus e feedbacks básicos.
